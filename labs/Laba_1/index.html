    Создание статического сайта

    Сайт: https://annapanfilova2007.github.io/
    Репозиторий GitHab: https://github.com/AnnaPanfilova2007/annapanfilova2007.github.io

    Цель

    Освоение основных принципов создания статического сайта с помощью генератора MkDocs и его развертывание на платформе GitHub Pages.

    Задачи

    • Установить и настроить программное обеспечение (Python, MkDocs).

    • Изучить структуру проекта MkDocs и принципы организации файлов.

    • Освоить синтаксис Markdown для форматирования текстового контента.

    • Создать базовую структуру сайта с несколькими страницами.

    • Настроить навигацию между страницами через файл mkdocs.yml.

    • Установить и подключить тему Material для MkDocs.

    • Выполнить кастомизацию внешнего вида сайта с помощью CSS.

    • Научиться запускать локальный сервер для разработки (mkdocs serve).

    • Развернуть готовый сайт на платформе GitHub Pages.

    • Оформить отчет о проделанной работе.
