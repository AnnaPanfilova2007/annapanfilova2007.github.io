## 🧭 О чем этот проект

Этот проект — документация/блог/портфолио, созданное с помощью **MkDocs**. Основная цель — предоставить удобную и структурированную информацию о [вставьте тему: разработке, учебе, технологиях, путешествиях и т.д.].

Мы стремимся к тому, чтобы контент был:
- понятным;
- актуальным;
- полезным;
- красиво оформленным.

---

## 🛠️ Технологии

Проект построен на современном стеке:

- [MkDocs](https://www.mkdocs.org/) — генератор статической документации;
- [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) — тема, которая делает документацию красивой;
- [Python](https://www.python.org/) — язык, на котором всё это работает;
- Markdown — язык разметки для написания контента.
